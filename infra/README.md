# Infra Roadmap

This directory is reserved for the platform layers that follow the local Compose MVP:

- `helm/aws-collaboration-dashboard`: future application chart for k3d and k3s
- `opentofu`: infrastructure definitions for the Hetzner VPS, firewall, and DNS handoff

The current scaffold keeps these folders in place so app and API work can grow into the later deployment phases without restructuring the repo.
